(static) createPopup(group, titleopt, contentsopt, controlleropt, paramopt) → {Popup}

ポップアップを作成

作成した時点では表示されません。

第1引数にはグループ名を指定します。グループ名は同じグループのポップアップは同時に1つだけ存在します。 (同一グループのものを開くと、現在開いているものが消えます。)
