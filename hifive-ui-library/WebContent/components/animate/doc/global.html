Parameters:
Name Type Attributes Default Description
element jQuery アニメーション対象の要素
option.duration string | number <optional>
 "normal" アニメーションの動作期間。"slow","normal","fast"または数値でミリ秒指定
option.easing string <optional>
 アニメーションの変化量を調節するカスタム関数名
option.isTransition Object <optional>
 アニメーションの動作をCSSトランジションを使用するか。使用しない場合はjQuery.animate()を使用します。 デフォルトはCSSトランジションを使用します。ただしCSSトランジションに対応していないブラウザではjQuery.animateを使用します。
option.complete Object <optional>
 アニメーションが終わった際に実行される関数
Returns:
アニメーションをストップするための関数を持つオブジェクト。以下のような2つの関数を持つオブジェクトを返します。 

{
	stop: fn, // アニメーションを即座に停止します。
	stopToEnd: fn
// アニメーション終了時のスタイルを適用してからアニメーションを即座に停止します。
